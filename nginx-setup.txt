## ğŸ› ï¸ Step 4: Create Nginx Server Blocks

### ğŸ”§ For `example1.com`

```bash

```

Paste:

```nginx

    listen 80;
    server_name example1.com www.example1.com;

    root /var/www/example1.com;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

### ğŸ”§ For `api.example1.com`

```bash

```

Paste:

```nginx
server {
    listen 80;
    server_name api.example1.com;

    root /var/www/api.example1.com;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

## ğŸ”— Step 5: Enable Sites

```bash
sudo ln -s /etc/nginx/sites-available/example1.com /etc/nginx/sites-enabled/
sudo ln -s /etc/nginx/sites-available/api.example1.com /etc/nginx/sites-enabled/
```

Optional: Remove default site

```bash
sudo rm /etc/nginx/sites-enabled/default
```
